# Pokemon Legendary Predictor

A machine learning web application built with Streamlit that predicts whether a Pokémon is Legendary based on its battle statistics.

## Features

* Interactive stat input
* Random Forest Classifier
* Prediction confidence score
* Feature importance visualization
* Pokémon type distribution analysis
* Top 10 strongest Pokémon dashboard

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Streamlit

## Dataset

Pokemon dataset containing 800+ Pokémon and their battle statistics.

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
